# RockPaperScissors
Small Approach for the Game Rock, Paper and Scissors

The Idea: The task involves implementing "Rock Paper Scissors" (https://en.wikipedia.org/wiki/Rock_paper_scissors) in Java.
When the program is run, two simulated players, "A" and "B," should compete against each other 100 times. 
While player "A" always plays "Paper," player "B" should randomly choose one of the three symbols for each turn.

This program is going to be implemented as a simple Java console application that runs without parameters and 
exits after 100 turns. The final result (wins/losses/draws) is printed to the console for each player
